# Notice
When downloading the HUD's zip file, the extracted folder **MUST** be named `fragmenthud`.

If it is not, then it will NOT work!

Your `<TF2 ROOT>/tf/` folder should look something similar to this:

```
.
├── ...
├── bin/
├── cfg/
├── custom/
│   ├── ...
│   ├── ...
│   ├── fragmenthud/
│   │   ├── ...
│   │   ├── base/
│   │   ├── extd/
│   │   ├── frag/
│   │   ├── info.vdf
│   │   └── ...
│   ├── ...
│   └── ...
├── maps/
├── resource/
├── scripts/
├── sound/
└── ...
```
